Kyzoku
======

Your basic shmup. It's just for practice using SDL. Yay!


Building Kyzoku
---------------
Requirements:
- SDL 1.2
- SDL_image 1.2
- SDL_ttf

GCC
---
- compile using the makefile in  /src/
- type 'make' and if all goes well, it should compile to 'kyzoku' which you can then run.


For CodeBlocks
--------------
- .cpb file included!



Controls
----------
Arrow Keys or WASD - Movement
C or J - Shoot
